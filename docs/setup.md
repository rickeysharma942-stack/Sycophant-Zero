# Sycophant-Zero: Installation & Deployment Guide

Welcome to the **Sycophant-Zero** setup guide. This document outlines the process for configuring your local environment to run the engine for asynchronous, persistent model interaction.

## 1. Prerequisites
Before starting, ensure your system meets the following requirements:
*   **Python 3.9 or higher**: Required for modern `asyncio` support.
*   **Git**: Required for cloning the repository.
*   **API Keys**: You must have active API keys for the providers you intend to use (e.g., OpenAI, Anthropic, or local LLM endpoints).

## 2.Configuration
 * make a file called api_keys.txt in the root folder.
